# Vibe Capturer - 빠른 시작 가이드

## 🚀 5분 안에 시작하기

### 1단계: API 키 발급 (2분)

1. [Google AI Studio](https://makersuite.google.com/app/apikey) 접속
2. Google 계정으로 로그인
3. **"Get API Key"** 또는 **"Create API Key"** 클릭
4. API 키 복사 (예: `AIzaSyC...`)

> 💡 **팁**: API 키는 무료이며, 매월 충분한 할당량이 제공됩니다.

### 2단계: 앱 열기 (30초)

Chrome 브라우저에서 아래 파일을 엽니다:
```
L:\obsidian auto\vibe-capturer\index.html
```

또는 파일 탐색기에서 `index.html`을 더블클릭하세요.

### 3단계: 설정 (1분)

1. 우측 상단 **⚙️ 설정** 버튼 클릭
2. **Google Gemini API 키** 입력
3. **폴더 선택** 버튼 클릭 → Obsidian 볼트 폴더 선택
4. **저장** 클릭

### 4단계: 첫 메모 만들기 (1분)

1. **마이크 버튼** 클릭 (마이크 권한 허용)
2. 생각을 **자유롭게 말하기**
   - 예: "삼성전자 김 과장님께 내일까지 견적서 보내기"
3. **중지 버튼** 클릭
4. AI가 자동으로 분류 및 마크다운 생성 (5~10초)
5. **"Obsidian에 저장"** 클릭

### 5단계: Obsidian에서 확인

Obsidian을 열고 방금 생성된 노트를 확인하세요!

## 🎯 사용 팁

### 효과적인 음성 메모 방법

✅ **좋은 예시:**
- "삼성전자 김철수 과장님께 다음 주 화요일까지 견적서 보내기"
- "오늘 10km 달렸는데 페이스 5분 30초, 오른쪽 무릎 통증"
- "FBA 사업에 AI 자동화 도구 접목 아이디어, 시장 조사 필요"

❌ **피해야 할 예시:**
- 너무 짧은 메모: "견적서"
- 배경 소음이 심한 환경
- 5분 이상의 긴 녹음

### 카테고리 자동 분류

- **영업** 키워드: 업체명, 담당자, 견적, 미팅, 계약 등
- **마라톤** 키워드: km, 달리기, 페이스, 무릎, 훈련 등
- **아이디어** 키워드: 생각, 아이디어, 인사이트, 전략 등

## ❓ 문제 해결

### "음성 인식을 지원하지 않습니다"
→ Chrome 브라우저를 사용하세요.

### "API 호출 실패"
→ API 키가 올바른지 확인하세요.

### "폴더 선택을 지원하지 않습니다"
→ Chrome 또는 Edge 브라우저를 사용하세요.

### 음성 인식이 안 됩니다
→ 마이크 권한을 허용했는지 확인하세요.
